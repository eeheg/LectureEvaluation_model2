navbar navbar-expand-lg navbar-light bg-light: 네비게이션 바의 스타일을 지정하는 클래스입니다.
navbar-expand-lg는 브라우저 크기가 큰 경우에만 네비게이션 메뉴가 확장되도록 설정하고, navbar-light는 밝은 배경색을 사용합니다.

navbar-toggler: 네비게이션 바의 작은 화면에서 햄버거 버튼을 나타내는 클래스입니다.

collapse navbar-collapse: 작은 화면에서 네비게이션 메뉴가 접혀서 보이지 않도록 하는 클래스입니다.

nav-item: 네비게이션 메뉴의 각 항목을 나타냅니다.

navbar-nav: 부트스트랩 내비게이션 바에서 사용되는 클래스로, 내비게이션 항목들을 나란히 정렬하는 역할을 합니다.

nav-link: 네비게이션 메뉴 항목 내 링크의 스타일을 지정하는 클래스입니다. 각각에 해당 클래스를 추가하여 스타일링합니다.

form-inline: 입력 필드와 버튼이 같은 줄에 나란히 표시되도록 하는 클래스입니다.

btn btn-outline-success my-2 my-sm-0: 버튼 스타일을 설정하는 클래스로, 초록색 테두리의 아웃라인 버튼 스타일을 나타냅니다.

container: 콘텐츠를 중앙에 정렬하고 여백을 설정하는 클래스입니다.

card bg-light: 카드 컴포넌트의 스타일을 지정하는 클래스입니다. bg-light는 밝은 배경색을 사용합니다.

card-header bg-light: 카드의 헤더 부분에 적용되는 스타일 클래스입니다. 마찬가지로 bg-light는 밝은 배경색을 사용합니다.

card-body: 카드의 본문 부분에 적용되는 스타일 클래스입니다.

btn btn-primary: 버튼 스타일을 설정하는 클래스로, 파란색의 일반 버튼 스타일을 나타냅니다.

role="dialog": 요소가 모달 창 역할을 한다는 것을 나타냅니다.

data-toggle="modal": 이 속성은 모달 창을 열기 위해 사용됩니다.

aria-labelledby: 모달 창의 라벨 역할을 하는 요소의 ID를 설정합니다.

aria-hidden="true": 스크린 리더에게 모달이 숨겨져 있다는 것을 알려줍니다.

tabindex="-1": 해당 요소를 키보드 탭 키로 접근할 수 없도록 설정합니다.
모달 창의 배경을 클릭해도 포커스가 창 내부 요소로 이동하지 않도록 하기 위해 사용될 수 있습니다.

modal fade: 모달 창의 스타일을 지정하는 클래스입니다. fade는 모달 창이 나타날 때 애니메이션 효과를 주는 역할을 합니다.

data-dismiss="modal": 클릭 시 모달을 닫는 역할을 합니다.

aria-label="Close": 스크린 리더가 "닫기"라는 라벨을 제공할 때 사용됩니다.

modal-dialog: 모달 창의 크기 및 위치를 설정하는 클래스입니다.

modal-content: 모달 내용을 감싸는 컨테이너입니다.

modal-header: 모달 창의 헤더 부분 스타일을 지정하는 클래스입니다.

modal-body: 모달 창의 본문 부분 스타일을 지정하는 클래스입니다.

form-group: 입력 필드와 레이블을 그룹화하는 클래스입니다.

form-control: 입력 필드를 스타일링하는 클래스로, 부트스트랩이 제공하는 일관된 스타일을 적용합니다.

btn-secondary, btn-danger: 버튼의 추가 스타일을 지정하는 클래스로, 각각 회색 테두리의 버튼과 빨간색의 버튼을 나타냅니다.

footer bg-dark: 페이지 하단의 푸터 부분의 스타일을 설정하는 클래스입니다. bg-dark는 어두운 배경색을 사용합니다.

text-center, text-left, text-right: 텍스트의 정렬 스타일을 설정하는 클래스로, 각각 가운데, 왼쪽, 오른쪽 정렬을 나타냅니다.

row, col-8, col-4, col-9, col-3: 그리드 시스템을 이용하여 레이아웃을 설정하는 클래스입니다.
row는 행(가로 줄), col-8는 8개의 칼럼을 차지하는 컬럼 등을 나타냅니다.
row 클래스는 일반적인 레이아웃에서 컨텐츠의 가로 배치를 그룹화하는 데 사용되고,
form-row 클래스는 폼 요소들의 가로 배치를 그룹화하는 데 사용됩니다.

bg-light, btn-outline-success, btn-primary, btn-danger, text-center, text-left, text-right
: 여러 요소에 공통적으로 사용되는 색상 및 스타일을 설정하는 클래스들입니다.

mx-1, my-2, mt-2: 여백(마진)을 설정하는 클래스로, mx는 가로 여백, my는 세로 여백, mt는 위쪽 여백을 나타냅니다.

my-sm-0: 작은 화면 크기에서 위아래 여백을 0으로 설정합니다. 즉, 세로 여백이 없습니다. (sm, md, lg, xl 등)

mr-auto: 요소를 오른쪽으로 정렬하고 나머지 여백을 자동으로 조절하여 왼쪽에 있는 내용과 오른쪽에 있는 요소 사이의 간격을 최대화하는 역할을 합니다.
항목들을 오른쪽으로 정렬하면서 나머지 여백을 자동으로 조절하여 화면의 크기에 따라 적절하게 배치되도록 돕습니다.

active: 네비게이션 메뉴 항목에 사용되는 클래스 중 하나입니다. 이 클래스는 현재 활성화된 항목을 나타내는 데 사용됩니다.
네비게이션 바에서 현재 사용자가 위치한 페이지나 섹션을 강조하거나 시각적으로 구분할 때 주로 사용됩니다.
네비게이션 메뉴에서 사용자가 현재 "메인" 페이지에 위치해 있다면, 해당 메뉴 항목에 active 클래스를 추가하여
사용자에게 현재 위치를 시각적으로 알려줍니다.
즉, "메인" 메뉴 항목이 사용자가 현재 위치한 페이지를 나타내는 항목임을 나타냅니다.

aria-label="Search": 웹 접근성을 위해 스크린 리더(Screen Reader)가 해당 요소를 설명할 때 사용할 텍스트를 지정합니다.
이 경우, 검색 입력 필드를 나타내며 "검색"이라는 라벨을 제공합니다.
